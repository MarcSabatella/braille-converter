Braille Converter

This tool converts from alphanumeric input representing dot patterns like "1456 156 1234" or "fjkl fkl fdsj" into Unicode. It can also convert from braille ASCII like "?:P".

For dot patterns:

  • Numbers represent dots using the standard 1-6 convention
  • letters are as with the six-key input system (fdsjkl for 123456)
  • 0 or semicolon (;) represent a blank cell

After pressing one of the Convert buttons, the resulting Unicode braille can be copied and pasted into a text editor.
